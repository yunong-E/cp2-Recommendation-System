- 평가지표가 서로 다를 때(`MAE`, `RMSE`와 같이) 어떤 평가지표를 믿으면 좋을까요? ex. 수식이 어려운 것?(말이 안됨). 평가지표에 대한 `신뢰성` 부분까지 생각할 것. 
- 평가지표를 현장에 적용을 시켜야 할것. 현장에 적용하게 되었을 때 좀 더 유의미하게 영향을 미치는 지표. 미미하지만 조금 더 영향을 미치는 지표. 
- 여러가지 후보군에 대해 영향지표를 같이 표기하는 것이 좋다. 현장에 계신 분들에게 토스를 하는 것이다(우리가 풀 수 있는 문제에 대해서 제시를 할게, 너희들의 현장에 맞는 지표와 도메인 지식에 맞게 토스를 할게).
- 보통 평가지표 그래프는 연구결과 보고서 `끝단`에 있다. 여러 평가지표를 함께 표기한다. 결론은 `여러 평가지표(결과)를 함께 사용`하라.
- 뛰어난 인사이트, 모델링 하는 것도 좋지만, 일정을 맞추는 것이 가장 중요하다.[`데드라인`] 하루, 이틀정도 여유있게 완성하는 것이 좋습니다.
